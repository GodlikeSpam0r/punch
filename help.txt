Usage

$ punch

    Print current month. Today is highlighted in blue, the last updated day in
    green.

$ punch <start>-<end>*

    Add blocks to today. E.g.: punch 8-12 12:30-18

$ punch -e (--edit)

    Edit current BRF file with your default text editor.

$ punch -h (--help)

    Print this message.

$ punch -h (--hack)

    Edit this script.
