Usage

$ punch

    Print current month. Today is highlighted in blue, the last updated day in
    green.

$ punch HH:MM-HH:MM HH:MM-HH:MM ...

    Add blocks to today. Minutes are optional. e.g. punch 8-12 12:30-18

$ punch -d, --day DD.MM.YYYY

    Pass a date to add blocks for that day. Defaults to today.
    e.g. punch -d 03.12.2014 22-23

$ punch -e, --edit

    Edit current BRF file with your default text editor.

$ punch -f, --format

    Reformat current BRF file. This will normalize whitespace.

$ punch -h , --help

    Print this message.

$ punch -H, --hack

    Edit this script.

$ punch -r, --raw

    Print current BRF file without fancy highlighting and auto-formatting.

$ punch -u, --update

    Update this script. (Given you installed it via git-clone)
